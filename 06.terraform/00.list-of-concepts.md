# Terraform Course

## Course Objective
Will design, provision, and manage production-ready AWS infrastructure using Terraform with CI/CD integration.

---

# Module 1 – Terraform Fundamentals & Core Workflow

## Concepts
- Infrastructure as Code (IaC)
- Imperative vs Declarative
- Why Terraform?
- Terraform Architecture (Core, Providers, Plugins)
- Terraform CLI workflow:
  - terraform init
  - terraform plan
  - terraform apply
  - terraform destroy
- HCL basics (syntax, blocks, arguments)
- Provider & Resource blocks
- Implicit dependencies

## Practical
- Install Terraform
- Configure AWS CLI
- Create EC2 + Security Group
- Destroy infrastructure

# Module 2 – Variables, Outputs & Expressions

## Concepts
- Input variables

- tfvars files
- Output values
- Sensitive outputs
- Expressions
- Built-in functions
- Conditional expressions

## Practical

- Implement conditional resource creation

- Backend configuration

- Meta arguments:

- Dynamic blocks
