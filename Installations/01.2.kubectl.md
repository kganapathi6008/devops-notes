# Installing and Configuring kubectl (Windows)

## Step 1: Install kubectl on Windows

`kubectl` is the command-line tool for interacting with Kubernetes clusters. You can install it on Windows using any of the methods below.

---

### **Method 1: Install via the Official Kubernetes Release**

1. **Download the latest kubectl.exe** (Windows x86-64):

   Official Kubernetes docs:
   [https://kubernetes.io/docs/tasks/tools/install-kubectl-windows/](https://kubernetes.io/docs/tasks/tools/install-kubectl-windows/)

   Or download directly:

   ```powershell
   curl.exe -LO "https://dl.k8s.io/release/v1.31.0/bin/windows/amd64/kubectl.exe"
   ```

2. **Move kubectl.exe to a directory in your PATH**, for example:

   ```powershell
   move .\kubectl.exe "C:\Program Files\kubectl\kubectl.exe"
   ```

3. **Add the directory to your PATH** (if not already):

   * Open **System Properties → Advanced → Environment Variables**
   * Edit **Path**
   * Add:

     ```
     C:\Program Files\kubectl
     ```

4. **Verify the installation:**

   ```powershell
   kubectl version --client
   ```

---

### **Method 2: Install using Chocolatey**

If you have Chocolatey installed, run:

```powershell
choco install kubernetes-cli -y
```

Verify:

```powershell
kubectl version --client
```

---

### **Method 3: Install using Winget (Windows Package Manager)**

```powershell
winget install -e --id Kubernetes.kubectl
```

---

## Step 2: Configure kubectl

`kubectl` uses a kubeconfig file located at:

```
C:\Users\<your-username>\.kube\config
```

If you receive a kubeconfig file from your cluster administrator or cloud provider, place it in this directory.

Verify your cluster connection:

```powershell
kubectl get nodes
```
